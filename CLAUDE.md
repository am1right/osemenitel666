# Правила работы (всегда)

- Не перепроверять код в браузере без явной просьбы.
- Не читать файлы целиком — только нужные куски (grep / offset+limit).
- Минимум прозы: без преамбул, без резюме, без пересказа сделанного.
- После задачи — только «готово» + список затронутых файлов.
- Минимизируй трату токенов настолько насколько это возможно без вреда для разработки.
